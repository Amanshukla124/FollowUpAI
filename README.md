# 🚀 FollowUpAI

**FollowUpAI** is an AI-powered communication assistant that helps users generate professional emails from meeting transcripts and create personalized cold outreach emails in seconds.

---

## ✨ Features

### 🧠 AI Follow-up Email Generator

* Convert meeting transcripts into structured emails
* Includes:

  * Summary of discussion
  * Action items
  * Next steps
* Saves hours of manual work

### 📩 Cold Email Generator

* Generate personalized outreach emails
* Useful for:

  * Internship applications
  * Sales outreach
  * Networking

### 🎯 Email Templates

* Pre-built templates for:

  * Follow-ups
  * Cold outreach
  * Professional communication
* Customizable tone (formal, friendly, persuasive)

### 📚 History Dashboard

* Stores previous transcripts and generated emails
* Easy access and reuse

### 📋 Copy to Clipboard

* One-click copy for quick usage in Gmail or other platforms

### 🎨 Modern UI

* Clean and responsive interface
* Designed for productivity and ease of use

---

## 💡 Problem It Solves

After meetings or calls, professionals spend 10–15 minutes writing follow-up emails.

FollowUpAI reduces this to **30 seconds** by automating:

* Email drafting
* Structuring content
* Highlighting key points

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **Database:** SQLite
* **AI Integration:** LLM API (OpenRouter / OpenAI compatible)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/FollowUpAI.git
cd FollowUpAI
```

### 2. Create virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install flask openai
```

### 4. Add API Key

Open `app.py` and replace:

```python
api_key="YOUR_API_KEY"
```

### 5. Run the app

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```

## 🧪 Usage

1. Paste your meeting transcript
2. Click **Generate Email**
3. Copy and use the generated email

OR

1. Enter details for cold outreach
2. Generate personalized email
3. Send directly via your email client
